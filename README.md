MedVault is a role-based healthcare management system developed using **Java Spring Boot**.  
It enables secure and efficient interaction between **Patients**, **Doctors**, and **Admins** for appointment booking, medical records management, and doctor verification.

---

## Features

### Patient Module
- User registration and login
- Book appointments with doctors
- Select available time slots
- View prescriptions and medical records
- Upload medical documents
- Send emergency requests
- Rate and review doctors

### Doctor Module
- Registration with document submission
- Admin verification required for activation
- Manage appointment slots
- View patient details and records
- Add prescriptions
- View ratings and feedback

### Admin Module
- Verify and approve or reject doctor registrations
- Manage doctors and patients
- Monitor system operations
